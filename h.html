Teori Algoritma Naive Bayes

Naive Bayes adalah algoritma klasifikasi berbasis probabilistik menggunakan Teorema Bayes dengan asumsi independensi kondisional antar fitur.

Rumus Naive Bayes
P(C | X) ∝ P(C) × ∏ P(xᵢ | C)
Di mana: C = kelas (Layak / Tidak Layak) X = vektor fitur input P(C) = prior probability kelas C P(xᵢ|C) = likelihood fitur xᵢ pada kelas C
Laplace Smoothing
P(xᵢ | C) = (count(xᵢ, C) + α) / (count(C) + α × |Vᵢ|)
α = 1 (parameter smoothing) |Vᵢ| = jumlah nilai unik untuk fitur ke-i count(xᵢ, C) = frekuensi xᵢ pada kelas C
Normalisasi Posterior
P(C | X) = score(C) / Σ score(C') untuk semua C'
Sehingga P(Layak|X) + P(Tidak|X) = 1.0 (100%)
Langkah Perhitungan
LangkahProsesFormulaKeterangan
1Hitung Prior(count(C)+1) / (N+|C|)Dengan Laplace Smoothing
2Hitung Likelihood(count(xᵢ,C)+1) / (count(C)+|Vᵢ|)Per fitur per kelas
